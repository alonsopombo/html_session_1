  • input element - dependiong on the type selected it will change, no closing tag
  • placeholder - always put there so it shows inside the box what it is
  • label - make forms easier to use, and links text to a form for screen readers, must give it an id and for, only one element one id
  • buttons - can be inside a form or not inside a form, if inside a form it will submit the form, unless a type attribute is given
  • using input type submit, can also create a button, but not common
  • Name attribute - to tell it what exactly to look for
  • Any input inside a form without a button, when click enter still submits
