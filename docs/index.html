Playing with semantic elements

  • main - represents main content of the page
  • nav - to create navigation menu
  • section - to mention a particular section
  • article - any independent composition
  • aside - not essential info like sidebars, or callouts boxes
  • header & footer
  • time - to specify dates
  • figure - for images
